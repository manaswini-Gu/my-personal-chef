## 🤖 Project: Personal Chef Agent

In this project, I created an AI agent that:

* Takes ingredients as input
* Uses a web search tool (Tavily)
* Finds relevant recipes
* Suggests dishes and can provide instructions

This helped me understand how:

* Models work
* Tools are integrated
* Agents make decisions

---

## 🧠 Key Learnings

* LangChain helps in building real-world AI applications
* Agents combine **LLMs + tools + instructions**
* Prompt design is very important
* Environment setup and API handling are crucial
